# 三大资源可视-前端

## 如何开发
克隆代码库，进入目录
```bash
$ git clone http://10.124.128.2:8888/CustOperPlatform/TaskScheduling/task_scheduling_ui.git
```
分支选择
```bash
ts_15664_20220101
```

安装初始化环境
```bash
$ npm install
```

进入开发模式
```bash
$ npm start
```

## 开发环境
- node版本 >= 12
